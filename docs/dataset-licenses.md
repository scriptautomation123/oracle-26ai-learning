# Dataset licenses and usage notes

- **PaySim** (`ealaxi/paysim1`, Kaggle): documented as CC BY; verify current Kaggle dataset page before production use.  
  Link: <https://www.kaggle.com/datasets/ealaxi/paysim1>
- **LendingClub** (`wordsforthewise/lending-club`, Kaggle): usage governed by Kaggle Terms of Service and dataset page constraints.  
  Link: <https://www.kaggle.com/datasets/wordsforthewise/lending-club>
- **Banking77** (`hwassner/banking77` on Kaggle or `PolyAI/banking77` on HuggingFace): CC BY attribution requirements apply.  
  Links: <https://www.kaggle.com/datasets/hwassner/banking77>, <https://huggingface.co/datasets/PolyAI/banking77>
- **UCI Bank Marketing** (UCI repository): publicly shared research dataset; follow UCI citation guidance.  
  Link: <https://archive.ics.uci.edu/dataset/222/bank+marketing>

Always validate current license text at source before commercial deployment.
